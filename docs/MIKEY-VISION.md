# Mikey Vision — 10/10 Product

Lead enters from any source
→ Mikey normalizes it into one clean lead
→ Mikey enriches it with project, campaign, buyer, source, inventory, agent, consent, and history context
→ Mikey scores urgency and intent
→ Mikey chooses the best next action
→ Permission gate checks risk, consent, quiet hours, WhatsApp window, owner rules, and autonomy mode
→ Mikey executes or asks approval
→ WhatsApp/call/email/SMS/agent task/site visit/cost sheet/payment action happens
→ Every action is logged
→ Every result comes back through webhook or agent update
→ CRM updates itself automatically
→ Mikey schedules the next follow-up
→ Mikey alerts owner only when needed
→ Mikey learns from the outcome
→ Next decision becomes smarter

## Build Order

1. **Universal Lead Intake** — every source becomes one standard lead object: WhatsApp, form, portals, Facebook, Google, walk-in, channel partner, import.

2. **Lead Context Builder** — for every lead, Mikey gathers: buyer details, source, campaign, project, budget, location, previous history, inventory match, consent, agent availability, current stage.

3. **Intent + Urgency Scoring** — Mikey decides if the lead is hot, warm, cold, duplicate, junk, investor, broker, or serious buyer.

4. **Next Best Action Engine** — Mikey chooses: call now, WhatsApp now, send brochure, ask question, book visit, assign agent, start nurture, wait, or ask approval.

5. **Permission + Safety Gate** — checks: autonomy mode, opt-out, quiet hours, WhatsApp 24-hour rule, risky action, discount/payment/legal approval, daily limits, owner rules.

6. **Action Execution Layer** — one common executor for: WhatsApp, SMS, email, voice call, IVR handoff, task creation, visit booking, unit match, cost sheet, payment reminder.

7. **Voice Agent Loop** — Dograh/Riya calls the lead, asks questions, handles objections, transfers to human when asked, sends transcript/outcome back to Mikey.

8. **Result Listener** — captures: delivered, read, replied, failed, answered, missed, busy, transferred, visit booked, no-show, cost sheet viewed, payment clicked.

9. **CRM Auto-Update** — Mikey updates: stage, notes, timeline, buyer requirements, lead score, agent task, source health, next follow-up.

10. **Follow-Up Scheduler** — Mikey automatically creates the next step: 5 min retry, 2 hour callback, tomorrow reminder, site visit reminder, no-show recovery, payment reminder, nurture campaign.

11. **Approval Inbox** — Owner sees: "Mikey wants to send this", "Mikey wants to call these 12 leads", "Mikey found a discount request", "Approve all / edit / reject".

12. **Owner Command Center** — one page shows: today's leads, hot leads, stuck leads, missed follow-ups, site visits, bookings, payments, campaign ROI, source health, salesperson performance.

13. **Campaign Brain** — Mikey creates WhatsApp campaigns: new launch, price revision, site visit reminder, no-show recovery, dead lead revival, payment follow-up, festive offer, CP broadcast.

14. **Revenue Attribution** — every source and campaign shows: leads, contacted, replied, visits, bookings, revenue, cost, ROI.

15. **Learning Memory** — Mikey remembers: what message worked, which agent converts, what buyer type books, what source is junk, what time gets replies, what project has demand.

16. **Self-Healing Monitoring** — Mikey checks every few minutes: WhatsApp failed, portal leads stopped, webhook replay needed, agent inactive, source dropped, payment failed, campaign underperforming.

17. **Audit + Trust Layer** — every action has: who/what/when/why, input context, decision reason, approval status, result, rollback option.

18. **Production Proof** — must pass: backend tests, frontend tests, webhook replay tests, mobile tests, one full browser E2E journey, one synthetic live journey every day.

19. **Pilot Mode** — for first client: observe for 7 days, recommend only, then approve-before-send, then limited autonomy, then full autonomy for low-risk actions.

20. **Final Feeling** — The owner should feel: "I do not need to chase my team every hour. Mikey saw the lead. Mikey contacted them. Mikey updated the CRM. Mikey booked the visit. Mikey warned me where humans are needed. Mikey is running the sales floor."
