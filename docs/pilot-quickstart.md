# Pilot Quick-Start Checklist

> Print this page. Tick each box as you complete it on day 1.

## Team Setup
- [ ] **Create team accounts** — Settings → Team → Add Users
      (Owner, Admin, Sales Agent roles)
- [ ] **Each user logs in** — verify they see the dashboard
- [ ] **Set permissions** — restrict access if needed (e.g. agents see only
      their leads)

## WhatsApp
- [ ] **Configure provider** — Settings → Integrations → WhatsApp
      (Cloud API or third-party)
- [ ] **Submit templates** — at least: lead follow-up, visit reminder,
      payment due notice
- [ ] **Send test message** — from any lead conversation

## Projects & Units
- [ ] **Add projects** — Projects → Add Project (name, location, config)
- [ ] **Import units** — Projects → [Project] → Units → Bulk Import
      (use the CSV template)

## Leads
- [ ] **Export leads** from your old system / Excel as CSV
- [ ] **Import leads** — Imports → Import Leads → upload → map columns
- [ ] **Verify** — check 3–5 imported leads have correct names, phones,
      sources

## Portal Sources
- [ ] **Connect MagicBricks / 99acres / others** — Settings → Integrations
      → Portal Sources
- [ ] **Enable auto-import** — new leads from portals flow in automatically
- [ ] **Tag source** — verify a test lead shows the correct source label

## WhatsApp Templates
- [ ] **Templates approved** by WhatsApp (may take 24–48h)
- [ ] **Map templates** — Settings → Message Templates → assign to
      workflows (lead follow-up, visit confirm, payment reminder)

## Mikey Autonomy
- [ ] **Set autonomy level per category** — Mikey → Autonomy tab
      (Autonomous / Observe / Off for each)
- [ ] **Review learned rules** — check the "Learned Rules Awaiting Review"
      section
- [ ] **Set quiet hours** — when should Mikey NOT send messages?

## Channel Partners
- [ ] **Invite first partner** — Channel Partners → Add Partner
- [ ] **Generate referral link** — share with partner
- [ ] **Verify referral tracking** — lead from partner link shows source

---

## Day 1 Complete?

If all boxes are ticked, your CRM is set up with real data, connected to
WhatsApp and portals, and configured for autonomous operation. The team can
start working from the system on day 2.

**Need help?** Check the [User Guide](./user-guide.md) or runbooks in
`docs/runbooks/`.
